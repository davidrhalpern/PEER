# Security Policy

## Supported Versions
This repository is currently in an early, pre-release state. No production deployment is supported at this time.

## Reporting a Vulnerability
If you believe you have found a security vulnerability in this repository or its future implementation:
- Do not open a public Issue with exploit details.
- Instead, contact the repository owner privately with a clear description and any relevant reproduction steps.

## Unclassified-only reminder
Do not include classified information in any report or discussion.
